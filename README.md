¿Qué hace el programa?

Este programa tiene como objetivo calcular la cuota mensual de un préstamo. El usuario debe ingresar los datos principales para que el sistema pueda realizar los cálculos correspondientes.

 ¿Qué datos recibe de entrada?

El programa recibe como datos de entrada el monto total del préstamo, la tasa de interés anual expresada en porcentaje y la cantidad total de cuotas o meses en los que se va a pagar.

. ¿Qué datos procesa?

Con estos datos, el sistema procesa la información aplicando la fórmula de amortización para determinar el valor de cada cuota y también calcula el total de intereses que se generarán durante toda la vida del préstamo.

Qué datos imprime?

Al finalizar, el programa imprime en pantalla el valor de cada cuota mensual, el monto total a pagar por el préstamo y una tabla de amortización detallada.
